Color Website

RGB

This is a simple demonstration of how changing the red, green, and blue values produces different colors. Make sure that the different values are between 0 and 255. Using 0 for all values will give you black, and 255 for white.

HSL

This is a simple demonstration of how changing the hue, saturation, and lightness values produces different colors. Make sure that the hue value is between 0 and 360, and the saturation and lightness values are a percentage.
